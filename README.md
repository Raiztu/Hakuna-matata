# Hakuna-matata
Игра «Виселица», предположительно, 2 состояния: меню и сама игра.

В главном меню, на фоне тетрадного листа, располагаются название игры, кнопки «Играть» и «Выход», а также кнопка включения/выключения «Звука».

Основная страница игры - тот же фон, что и в главном меню, на экране будут кнопки-буквы и при неправильном выборе букв будет отрисовываться виселица.
